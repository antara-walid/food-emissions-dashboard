Foodprint

Bienvenue sur Foodprint

Ce projet permet d'explorer l'impact de systèmes alimentaires sur les émissions de gaz à effet de serre (GES). L'objectif est de réaliser une application interactive permettant de déconstruire l'empreinte carbone de différents aliments (viandes, céréales, laitiers, etc.) à travers leur cycle de vie complet.

Sélectionnez une page dans le menu ci-dessus pour commencer l'exploration.
